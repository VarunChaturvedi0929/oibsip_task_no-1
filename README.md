# 📚 StudyAI

StudyAI is a modern and responsive AI-powered educational landing page built using HTML, CSS, and JavaScript. It is designed to provide students with an engaging platform to explore AI-based learning tools.

## 🚀 Features

- Modern and clean UI
- Fully responsive design
- Attractive Hero section
- Interactive navigation bar
- Call-to-action buttons
- Fast loading and lightweight

## 🛠️ Tech Stack

- HTML5
- CSS3
